# 狩猎难觅其踪的个人网站

个人网站大部分都隐藏在互联网的丛林中，是一种有点神秘的动物，很少有人能发现它们。那些完全生活在 Facebook 和 Google 的黄金围墙花园中的没什么经验的互联网用户甚至可能不知道这个物种的存在，其它经验较少的用户或许认为个人网站已经像雷龙一样灭绝了，要么认为它像雪人或者大脚怪一样是个神话。但事实上，在野外还存活着许多个人网站，为了成功捕获它，猎人需要很好地了解这个物种的习惯。
