# 小小萌介绍

小小萌是归属于上海大学学盟社的群聊机器人 bot，目前基于 none-bot 与 mirai 两个框架进行开发（嫖插件）。功能方面的话目前还在继续完善更新，欢迎大家来提建议哦！

# 词库回复

为了更好的在学盟社团群聊中为新同学答疑，小小萌内置了新生常见问题，并通过关键词（头部 | 尾部 | 模糊 | 精准等）来自动回复，目前词库如下：

小小萌回复词库

1. 小小萌使用手册
2. 关于学盟
3. 上大网址导航
4. 上大校历
5. 校园地图
持续更新 ing

# 猜单词 wordle

在每天清晨 5—6 点与晚上 11—12 点（假期额外开放下午 1 点至下午 2 点），小小萌开放猜单词功能，帮助同学们在玩中学习。
玩法说明：
发送关键词游戏开始后，小小萌会发送单词格图片，玩家说出单词，格子中绿色代表字母位置正确，黄色代表存在这个字母但位置不对，灰色则代表不存在这个单词，在有限步内说出正确单词即为胜利。
