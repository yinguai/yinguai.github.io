# 使用说明

使用提示：部分网页浏览时需要挂上海大学 VPN（ 关于上大 VPN 使用：https://vpn.shu.edu.cn/index.htm ）

本网站为个人兴趣制作，欢迎在评论区留下你的宝贵建议，也欢迎来加入上海大学学盟社 Q 群：823547018。ヾ (≧▽≦*) o

# 常用功能

# 上大官网

介绍：上大官方网站，不过一般用的不太多
链接：https://www.shu.edu.cn/

# 成绩查询

介绍：主要用于考试周查成绩，不过一般这个时候学校网站总会崩溃，建议提前进
链接：https://cj.shu.edu.cn/StudentPortal/ScoreQuery

# 选课网站

介绍：上大一年要进（蹲课）n 次的网站，选课前要好好了解下自己的教学计划书哦
链接：http://www.xk.shu.edu.cn/

# 排课助手

介绍：外置的选课小助手，在正式开放选课前可以在上面先预排课表，实测好用
链接：https://xk.shuosc.com/

# 查询空教室

介绍：用来在平时没课时去空教室自习的网站，不过也要注意在空教室也不要影响到其他人哦
链接：https://cj.shu.edu.cn/RoomUse/RoomUseDate/

# 物理实验中心

介绍：注意选实验时间，不然就要随机分配了
链接：https://www.phylab.shu.edu.cn/

# 图书馆书籍查询

介绍：线上图书馆查阅资料和文献用的，平时使用会较少
链接：http://202.120.117.231:8991/F/YP82FYYBJHBUMFXQSY6NRQGC898HCNJVK8M41AQAL1I169KJQG-08364?&pds_handle=GUEST

# 校园邮箱

介绍：上大官方邮箱在此（虽然平时也基本用不到），但还是记一下自己的密码与账号，以防需要时使用
链接：http://newmail.shu.edu.cn/

# 上大网课

介绍：可以看上大网课，不过现在一般都直接用学习通了
链接：http://www.elearning.shu.edu.cn/portal

# 一网通办

# 本科新生报到

# 报到须知

介绍：官方文档还是很全面的，可以认真阅读下提前了解信息
链接：https://yingxin.shu.edu.cn/info/1007/1373.htm

# 信息登记

介绍：新生开学要做事项，要注意下时间节点哦
链接：https://yingxin.shu.edu.cn/bksyxrk.htm

# 校园地图

介绍：顾名思义，上大还有几张手绘的地图，也是很好看的
链接：https://map.shu.edu.cn/

# 团关系转接

介绍：开学转接团关系时要用到的，不记得账号密码可以问下高中团支书
链接：https://zhtj.youth.cn/zhtj/

# 班级分配

链接：https://eportal.shu.edu.cn/user/

# 宿舍分配

链接：https://yingxinweb.shu.edu.cn/BenKS/XinXCX.aspx

# 身份激活

介绍：激活仅第一次就可以了的
链接：http://services.shu.edu.cn/

# 社区学院

介绍：大类学生大一所归属的学院，不过一般也用的不太多，公告与活动通知可以了解下
链接：https://residential.shu.edu.cn/

# 户口迁移须知

介绍：这个主要是关于落户上海政策的，有需求的 UU 可以看下
链接：https://yingxin.shu.edu.cn/info/1008/1261.htm

# 校园生活

# 校医院

链接：https://www.xyy.shu.edu.cn/

# 学生卡副卡补办

链接：https://pim.shu.edu.cn/infoplus/form/3441962/render

# 课外活动查询

介绍：由于上大有每个学期的课外活动次数要求，这个网站便是专门来查参加次数的（学校网站更新超级慢），学期结束前确保自己次数对了就 OK
链接：http://202.120.127.149:8989/spims/exercise.do?method=seacheload

# 专业信息公开

介绍：为大类学生准备的网站，大一一年可以多了解专业信息及培养计划，在分流时便可以不在盲目
链接：http://zyinfo.shu.edu.cn/

# 校历

介绍：众所周知，上大校历就是天（指非特别重大情况一般还照着校历上）
链接：https://www.shu.edu.cn/bzap.jsp?urltype=tree.TreeTempUrl&wbtreeid=1233

# 孰知网

介绍：评奖评优，申请奖学金以及记录成长经历的网站，较常用
链接：http://www.sz.shu.edu.cn/

# 英语四六级

介绍：上大不同英语分级学生在大一会对报考四级做限制，可以先提前了解下情况
链接：http://cet-bm.neea.edu.cn/

# 转专业申请

链接：https://zzy.shu.edu.cn/

# 安开网

介绍：大化课前要参加考试的网站，虽然很水不过要记得完成
链接：http://ankai365.shu.edu.cn/

# 计算机等级考

链接：https://eportal.shu.edu.cn/transfer/#id=174

# 上大教务处

链接：http://www.jwc.shu.edu.cn/
